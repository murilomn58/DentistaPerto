# DentistaPerto Homepage Redesign

## Date: 2026-02-28

## Summary
Complete homepage redesign including an accurate interactive Brazil SVG map with geographic boundaries, gradient heat map by dentist density, Framer Motion animations throughout, and visual upgrades to all homepage sections.

## Components

### 1. BrazilMap (Complete Rewrite)
- Accurate SVG paths for all 27 Brazilian states
- Gradient fill by dentist density (light cyan → deep blue)
- Hover: glow effect + tooltip (state name, dentist count, top city)
- Framer Motion staggered entrance animation
- Dark navy card background with subtle grid pattern

### 2. Hero Section (Enhanced)
- Animated dot pattern overlay
- Better typography hierarchy
- Frosted glass search bar
- Animated stats badge

### 3. Stats Section (Upgraded)
- Glass cards with gradient borders
- Animated counting numbers on scroll
- Gradient icon backgrounds

### 4. Map Section Layout (Redesigned)
- Full-width dark background with gradient mesh
- Premium card with glow border for map
- Ranked top cities list with population bars

### 5. Procedures Section (Polished)
- Gradient hover effects on cards
- Staggered scroll reveal

### 6. CTA Section (Enhanced)
- Gradient background with floating elements

## Dependencies
- Install: framer-motion
- Existing: tailwindcss, lucide-react, clsx, tailwind-merge

## Design Tokens
- Navy Dark: #0F172A
- Blue Primary: #2563eb
- Cyan Accent: #06b6d4 / #22D3EE
